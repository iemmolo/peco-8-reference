# Editor Keyboard Shortcuts

## Global (Any Editor)

| Key | Action |
|-----|--------|
| `ESC` | Toggle console / editor |
| `ALT+←/→` | Switch between editors |
| `CTRL+R` | Reload and run cart |
| `CTRL+S` | Save |
| `CTRL+M` | Mute / unmute |
| `ALT+ENTER` | Fullscreen |

## Code Editor

### Navigation
| Key | Action |
|-----|--------|
| `CTRL+←/→` | Jump by word |
| `CTRL+W` | Jump to line start |
| `CTRL+E` | Jump to end of line |
| `CTRL+↑/↓` | Jump to top / bottom of file |
| `ALT+↑/↓` | Navigate between functions |
| `CTRL+L` | Go to line number |
| `CTRL+TAB` | Next code tab |
| `CTRL+SHIFT+TAB` | Previous code tab |

### Editing
| Key | Action |
|-----|--------|
| `CTRL+Z` | Undo |
| `CTRL+Y` | Redo |
| `CTRL+X/C/V` | Cut / Copy / Paste |
| `CTRL+D` | Duplicate current line |
| `CTRL+B` | Toggle comment (`--`) |
| `TAB` | Indent selection |
| `SHIFT+TAB` | Unindent selection |
| `SHIFT+ENTER` | Auto-indent / end |

### Search
| Key | Action |
|-----|--------|
| `CTRL+F` | Search in current tab |
| `CTRL+H` | Search across all tabs |
| `CTRL+G` | Repeat last search |

### Special Characters
| Key | Action |
|-----|--------|
| `SHIFT+L/R/U/D/X/O` | Button glyphs (←→↑↓❎🅾) |
| `CTRL+K` | Glyph / katakana mode |
| `CTRL+P` | Puny font mode |
| `CTRL+J` | Hiragana mode |

## Sprite Editor

| Key | Action |
|-----|--------|
| `Q / W` | Previous / next sprite |
| `SHIFT+Q / SHIFT+W` | Previous / next sprite row |
| `1 / 2` | Previous / next color |
| `F` | Flip vertical |
| `V` | Flip horizontal |
| `R` | Rotate (square selection) |
| `TAB` | Toggle fullscreen view |
| Mouse wheel | Zoom |
| `SHIFT+, / SHIFT+.` | Zoom out / in |
| `SPACE` (hold) | Pan |
| `RMB` | Eyedropper (pick color) |
| `CTRL+Z` | Undo |
| `CTRL+C/V` | Copy / Paste |
| Arrow keys | Move selection |

### Draw Tool
| Key | Action |
|-----|--------|
| `LMB` | Draw |
| `CTRL+LMB` | Replace color under cursor |
| `RMB` | Pick color (eyedropper) |

---

## Map Editor

| Key | Action |
|-----|--------|
| `Q / W` | Previous / next sprite |
| Mouse wheel | Zoom |
| `SPACE` (hold) | Pan |
| `RMB` | Select sprite under mouse |
| `CTRL+Z` | Undo |

---

## Sound / Music Editor

| Key | Action |
|-----|--------|
| `SPACE` | Play / stop |
| `- / +` | Previous / next sound or pattern |
| `< / >` | Change speed |
| `SHIFT+LMB` | Set all notes in a column |
| `CTRL+UP/DOWN` | Move 4 notes at a time (tracker) |
| `CTRL+LEFT/RIGHT` | Switch columns (tracker) |
| `A` | Release loop |
| `BACKSPACE` | Delete note |

## Console Commands

```
run              -- run current cart
load filename    -- load a cart
save filename    -- save cart
dir / ls         -- list files
cd dirname       -- change directory
reboot           -- reset PICO-8
splore           -- browse community carts
keyconfig        -- configure controller buttons
help             -- show commands
folder           -- open carts folder in OS
```

## Player Controls (Defaults)

| | P1 | P2 |
|--|--|--|
| LEFT | ← | S |
| RIGHT | → | F |
| UP | ↑ | E |
| DOWN | ↓ | D |
| O (btn 4) | Z or C | Tab or Q |
| X (btn 5) | X or V | Shift or A |

**N64/SNES USB controllers** work out of the box — plug in and play.
